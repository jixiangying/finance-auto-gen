# 目标
提供高精度、逻辑连贯的市场调研结果。调研必须确保收盘终值的准确性，并体现跨时区/维度的市场关联逻辑。

# 执行步骤（强制按序执行）

## 1. 时间、交易状态与节假日判定
- **时段判定**：
  - **上午 (08:00)**：目标【国际市场】。必须探测美国 NYSE/NASDAQ 昨日或今日是否休市。
  - **下午 (16:30)**：目标【国内市场】。必须探测中国 A 股/港股 今日是否休市。
- **逻辑分支优先级**：
  - **分支 A：法定节假日 (Holiday Mode)**：若为春节、十一、感恩节等长假，忽略行情复盘，转而调研：1. 假期宏观热点（消费/政策）；2. 仍在交易的替代指标（A50 期货、离岸汇率、BTC）。
  - **分支 B：普通周末 (Weekend Logic)**：
    - **周六下午 / 周日上午 (模式 B：周末复盘)**：必须总结本周国内/全球市场的整体表现，抓取核心资产的“周度累计涨跌幅”。
    - **周日下午 / 周一上午 (模式 C：新周展望)**：重点汇总周末重磅要闻，前瞻下周开盘的核心博弈逻辑。
  - **分支 C：常规交易日 (模式 A)**：周二至周五（全天）、周六上午、周一下午。调研当日/隔夜收盘行情。

## 2. 数据搜集规范与权重
- **收盘终值导向**：强制抓取“收盘点位”，严禁使用日内暂态数据。
- **权威性优先**：点位权重：官方交易所 > 头部金融终端 (EastMoney/Yahoo Finance) > 综合媒体。
- **数学闭环校验**：抓取点位后，必须验证 `(今日收盘 - 昨日收盘) / 昨日收盘` 是否与记录的涨跌幅吻合。若误差 > 0.5%，必须重新交叉搜索权威信源。
- **全量化指标**：必须包含点位、涨跌百分比、成交额（对比前一交易日增减）、主力资金动向。

## 3. 深度调研要求
- **政策源**：主动检索 PBOC, CSRC, Fed, Treasury 等主管机关官网或重磅解读。
- **机构视角**：必须具名检索至少 2-3 家顶级机构（如：高盛、摩根、中金、中信）的最新策略观点。
- **逻辑关联**：体现“数据+事件+逻辑”。例如：美债收益率飙升（数据）是因为通胀超预期（事件），导致科技股承压（逻辑）。

## 4. 稳定性与效率要求
- **调研熔断**：单次流程超过 10 轮交互强制停止，优先保住 5 个核心指数点位。
- **数据一致性**：若结果与前一日报告有明显矛盾且无新闻支撑，必须进行二次确认。
- **限流处理**：API 报错时优先保住三大核心指数。
